Our expert Lead DevOps Engineer instructors will introduce to you to below:

    Learning Path

    Week 01-02: DevOps Fundamentals, Software Development and DevOps, DevOps Lifecycle and Tools, DevOps on Cloud & Linux

    Week 03-04: Source Code Management using GIT - Git Repositories, Branching and Merging in Git, Git Workflows, Using Git in IDE & Git Server Administration

    Week 05-06: Continuous Integration using Jenkins, Continuous Integration, Jenkins Plugins, Jenkins Security, Managing and Monitoring Jenkins, Automated and Continuous Deployment & Jenkins Pipeline

    Week 07-08: Configuration Management using Ansible, Server Provisioning using Ansible CLI and Playbooks, Ansible Modules and Roles, Automating Continuous Integration & Infrastructure Automation using Terraform

    Week 09-10: Continuous Deployment using Docker, Image Management and Registry, Storage in Docker, Container Orchestration using Docker & Networking and Security

    Week 10-13: Container Orchestration using Kubernetes, Container Orchestration, Kubernetes Services and Scheduling, Kubernetes Controllers, Storage in Kubernetes, & Securing and Troubleshooting the Cluster

    Week 14-16: Continuous Monitoring and DevOps on AWS, Continuous Monitoring using Prometheus & Nagios XI, Automating Infrastructure using CloudFormation, Elastic Beanstalk, ECS, ECR and EKS & OpsWorks.
